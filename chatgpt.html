기초 — ChatGPT 시작하기 입문

업데이트 · 2026-06-18

OpenAI가 만든 대화형 AI 어시스턴트입니다. 질문에 답하고, 문서·이미지를 읽고, 웹을 검색하고, 이미지를 생성하며, 음성으로 자연스럽게 대화하고, 코드를 작성·실행하는 일까지 한 화면에서 처리합니다. 무료로 시작할 수 있고, 필요에 따라 기능을 확장합니다.

접속

웹은 chatgpt.com ↗. iOS·Android 앱과 Windows·macOS 데스크톱 앱도 있어 어디서나 이어서 사용할 수 있습니다.

가입

이메일 또는 Google·Microsoft·Apple 계정으로 간편 가입. 무료(Free)로 바로 시작할 수 있으며 신용카드 필요 없습니다.

첫 대화 & 모델 선택

입력창에 자연어로 질문. 화면 위 모델 선택기에서 빠른 응답용 모델과 깊이 생각하는 추론(Thinking) 모델을 전환합니다.

기본 설정 (3분이면 충분)

⚙️ 설정 → 메모리(이름·직업·취향 기억), 맞춤 지침(예: "항상 한국어 존댓말로", "결론부터 먼저"), 데이터 관리(내 대화 학습 여부).

상단 모델 선택기 빠른 모델 vs Thinking 추론 모델 전환. 작업에 맞게 고르세요.

팁 — 맞춤 지침 예시: 설정 → 맞춤 지침에 "나는 한국어로 답해줘. 결론을 먼저 쓰고, 근거를 이어서 써줘. 코드는 주석을 한국어로." 같이 입력하면 모든 대화에 자동 적용됩니다.

핵심 기능 깊이 보기

업데이트 · 2026-06-18

ChatGPT는 단순 답변 AI가 아닙니다. 아래 5가지 기능을 알아두면 활용도가 크게 올라갑니다.

음성 대화 (Voice)

입력창 오른쪽 마이크 아이콘을 누르면 실시간 음성 대화가 시작됩니다. 고급 음성 모드는 자연스럽게 끼어들기, 감정 표현, 웃음소리도 인식합니다. 운전 중·요리 중에 손을 쓰지 않고 ChatGPT를 쓸 수 있습니다.

  • 모바일 앱에서 가장 편리하게 사용
  • 한국어 음성 인식·출력 지원
  • 회의 중 간단한 메모·번역·검색에 유용
📝 음성 입력 (예)
다음 주 월요일 오전 회의 안건을 영어로 번역해줘. "1. 2분기 매출 검토, 2. 신규 파트너 계약 논의"
✨ 결과 (영어 음성으로 읽어줌)

Meeting Agenda for Next Monday Morning
1. Q2 Sales Review
2. Discussion on New Partner Contracts

Canvas — 옆 창 공동 편집

Canvas를 열면 ChatGPT가 생성한 글·코드가 오른쪽 편집 창에 펼쳐집니다. 복사·붙여넣기 없이 특정 단락만 선택해 "이 부분 더 간결하게"라고 요청하거나, 직접 타이핑해 수정할 수 있습니다. 긴 보고서·기획안 작성에 매우 효과적입니다.

  • 부분 선택 → 인라인 수정 지시 가능
  • 코드의 경우 특정 함수만 리팩터링 요청
  • 내보내기(복사·다운로드)로 작업 완료
📝 입력
Canvas에서 2분기 성과 보고서 초안 작성해줘. 3개 섹션: 매출 요약 / 주요 성과 / 다음 분기 계획
✨ 결과

Canvas가 열리며 보고서 초안이 생성됩니다. 각 섹션을 드래그로 선택 후 "이 부분 좀 더 구체적으로"라고 수정 요청할 수 있습니다.

Deep Research

단순 검색이 아닙니다. ChatGPT가 수십 개 출처를 자동 탐색하고, 각주가 달린 리포트를 작성해줍니다. 시장 조사, 경쟁사 분석, 논문 리뷰 같은 작업을 몇 분 안에 처리합니다. Plus 이상 요금제에 포함됩니다.

  • 자동 웹 탐색 → 인용 출처 표시
  • 검색 과정·중간 결과 실시간 확인
  • 최종 리포트 다운로드 가능
📝 Deep Research 요청
2026년 글로벌 전기차 배터리 시장을 국가별로 비교 분석하고, 출처를 포함한 리포트를 작성해줘.
✨ 결과 (요약)

탐색된 출처 32개 기반 리포트:
• 중국: 글로벌 점유율 약 65%, CATL 독주 [출처: BloombergNEF 2025]
• 미국: IRA 보조금으로 국내 생산 급증 [출처: DOE 2026]
• 한국: 삼성SDI·LG에너지솔루션 유럽 진출 가속...

Custom GPTs (나만의 챗봇)

GPT 빌더를 열면 코드 없이 목적별 맞춤 챗봇을 만들 수 있습니다. 시스템 프롬프트, 파일 지식베이스, 외부 API 연결까지 설정 가능. 만든 GPT는 팀과 공유하거나 GPT 스토어에 배포할 수 있습니다.

  • 사내 규정·FAQ 기반 GPT 구축
  • 특정 글쓰기 스타일 전용 챗봇
  • 고객 응대·온보딩 자동화
📝 GPT 빌더 지시
항상 논문 심사위원 어조로 답해줘. 논리적 허점을 먼저 지적하고, 보완 실험을 구체적으로 제안해줘. 한국어로만 답해.
✨ 결과

'논문 심사 GPT'가 생성됩니다. 이후 이 GPT와 대화하면 항상 엄격한 학술 심사 관점으로 피드백을 받을 수 있습니다.

메모리 (Memory)

ChatGPT가 대화를 통해 나에 대한 정보를 기억합니다. 이름, 직업, 선호 언어, 말투 등을 자동으로 저장해 다음 대화에서 매번 설명하지 않아도 됩니다. 설정에서 언제든 확인·수정·삭제할 수 있습니다.

  • 한 번 알려주면 이후 대화에 자동 적용
  • 저장된 내용 직접 관리 가능
  • 원하지 않으면 메모리 기능 끄기 가능
📝 입력
나는 스타트업 마케터야. 항상 짧고 실행 중심으로 답해줘.
✨ 결과 (메모리 저장 후)

저장 완료: "사용자는 스타트업 마케터. 답변은 짧고 실행 가능한 형태로."
이제 모든 대화에 자동 반영됩니다.

Codex — 코딩 에이전트 중급

업데이트 · 2026-06-18

Codex는 단순히 코드를 알려주는 것이 아니라 실제로 코드를 읽고, 여러 파일을 수정하고, 명령을 실행해 작업을 끝까지 수행하는 에이전트입니다. 코딩을 잘 몰라도 "이 기능을 추가해줘"라고 말하면 알아서 처리합니다.

가장 빠른 시작 — CLI

npm i -g @openai/codex codex

처음 실행하면 로그인 안내가 뜹니다. ChatGPT 계정(Plus 이상에 Codex 포함) 또는 OpenAI API 키로 로그인합니다. macOS·Linux 정식 지원, Windows는 WSL 권장.

Codex 4가지 형태

CLI (터미널) 가장 빠른 시작. 대화형 실행·승인 모드로 자동 실행 범위 조절 (읽기만 / 수정 허용 / 전체 허용)
IDE 확장 (VS Code · Cursor) 에디터 안에서 직접 파일 읽기·명령 실행·수정. 작업 전후 Git 체크포인트 권장
Cloud / Web (chatgpt.com/codex) 격리 클라우드 샌드박스에서 작업. 변경사항 diff 검토 → 바로 Pull Request 생성
GitHub · Slack 연동 이슈 할당 또는 Slack 메시지로 Codex에 작업 위임. CI/CD 파이프라인 자동화

심화 — API · Function Calling 고급

업데이트 · 2026-06-18

더 깊게 활용하고 싶다면 OpenAI API와 Function Calling을 익혀보세요. 자신만의 AI 앱과 자동화 파이프라인을 만들 수 있습니다.

OpenAI API 시작하기

API 키 발급

platform.openai.com ↗ → 로그인 → API Keys → 새 키 생성. 발급 즉시 복사해두세요 (이후 다시 볼 수 없습니다).

환경변수 설정

터미널에 export OPENAI_API_KEY=sk-... 또는 .env 파일에 저장. 코드에 직접 넣지 마세요.

SDK 설치 후 첫 호출

Python: pip install openai / Node.js: npm install openai. 공식 퀵스타트에 5분 예제가 있습니다.

Responses API + Agents SDK 선택 (2026 신규)

새로운 프로젝트라면 레거시 Chat Completions 대신 Responses API + Agents SDK를 사용하세요. 멀티스텝 에이전트 구현이 훨씬 간단합니다.

Function Calling — AI에 실제 행동 능력 부여

AI 모델은 기본적으로 텍스트만 출력합니다. Function Calling을 쓰면 모델이 "이 함수를 이런 인자로 호출해야 한다"고 JSON으로 알려주고, 내 코드가 실제 실행한 뒤 결과를 다시 모델에 넘겨 최종 답을 완성합니다.

활용 예시

  • 예약 챗봇: 사용자가 "다음 주 화요일 2시로 예약해줘"라고 하면 모델이 create_booking(date, time) 호출
  • 재고 조회: "A 제품 재고 남았어?"에 check_inventory(product_id) 자동 실행
  • 슬랙 알림: 조건 충족 시 send_slack_message(channel, text) 호출로 자동 발송
더 깊은 API 활용은 심화 활용 페이지에서 Python SDK 예제, Agents SDK 아키텍처, 비용 최적화 팁까지 확인하세요.

자주 묻는 질문

업데이트 · 2026-06-21
무료 플랜으로 충분할까요?

가벼운 질문·번역·간단한 글쓰기라면 무료(Free)로 충분합니다. 다만 Deep Research, Canvas, Projects, Codex, 대용량 파일 분석은 Plus($20/월) 이상이 필요합니다. 먼저 무료로 써보고, 사용량 제한에 자주 걸리거나 고급 기능이 필요하면 업그레이드를 고려하세요.

어떤 모델을 골라야 하나요?

대부분의 일상 작업(이메일·번역·요약)은 기본 빠른 모델로 충분합니다. 수학 문제 풀이, 복잡한 코드 디버깅, 논리 추론이 필요한 작업은 Thinking(추론) 모델을 선택하세요. 입력창 위 모델 선택기에서 쉽게 전환할 수 있습니다. 추론 모델은 응답이 느리고 사용량을 더 소모합니다.

한국어로 잘 되나요?

네, 한국어 지원이 매우 잘 됩니다. 설정 → 맞춤 지침에 "항상 한국어로 답해줘"를 추가하면 별도 요청 없이도 한국어로만 응답합니다. 번역, 한국어 문체 교정, 한국 비즈니스 맥락을 반영한 글쓰기도 모두 가능합니다.

Plus($20)를 살 만한가요?

업무에서 주 5일 이상 사용한다면 충분히 가치 있습니다. Deep Research(시장 조사·리포트 자동 작성), Canvas(긴 문서 편집), Codex(코딩 에이전트), Projects(맥락 유지)를 쓸 수 있습니다. 시급 기준으로 봤을 때, 하루 30분만 절약해도 한 달 구독료를 넘깁니다.

내 대화 내용이 AI 학습에 쓰이나요?

기본 설정에서는 대화가 모델 개선에 활용될 수 있습니다. 원하지 않는다면 설정 → 데이터 관리 → '내 콘텐츠로 학습 개선' 해제를 선택하세요. Plus 이상 또는 Business/Enterprise 요금제에서는 기본적으로 데이터가 학습에 사용되지 않도록 옵션이 제공됩니다. 정확한 정책은 OpenAI 개인정보처리방침 ↗을 확인하세요.

Codex를 쓰려면 뭐가 필요한가요?

ChatGPT Plus 이상 계정이 있으면 Codex가 포함됩니다. CLI는 Node.js 설치 후 npm i -g @openai/codex로 시작합니다. Windows는 WSL(Windows Subsystem for Linux)이 권장됩니다. 코딩 지식이 없어도 자연어로 지시하면 되지만, 변경 결과를 검토하고 승인하는 능력은 도움이 됩니다.

ChatGPT와 Claude 중 뭐가 더 좋나요?

용도에 따라 다릅니다. ChatGPT는 이미지 생성, 음성 대화, 코딩 에이전트(Codex), 방대한 GPT 스토어 생태계가 강점입니다. Claude는 긴 문서 분석, 세밀한 글쓰기 품질, PC 파일 자동화(Cowork)가 강점입니다. 두 도구를 함께 쓰는 분도 많습니다. Claude 가이드도 참고하세요.

3. 개발

디버깅, 코드 리뷰, SQL·정규식, 테스트, 레거시 이해, 문서화 — 개발 생산성을 가장 직접적으로 올려주는 영역입니다. 핵심 원칙 하나: 맥락을 충분히 주면 답의 질이 달라집니다. 언어·버전·에러 전문·관련 코드를 함께 붙이세요.

버그 디버깅 Claude 중급

목표. 에러의 진짜 원인을 찾아 고치고, 같은 실수를 다시는 안 하기.  이럴 때 씁니다. 에러는 떴는데 원인을 모를 때 · 동작이 기대와 다를 때 · 남이 쓴 코드라 맥락을 모를 때.

1) 사전 준비 — 이 네 가지를 먼저 손에 쥐세요

  • 언어·버전 (예: Python 3.11 / Node 20 / Java 17)
  • 에러 메시지 전문 — 한 줄만 자르지 말고 스택 트레이스까지 통째로
  • 문제가 난 코드 + 그 코드를 호출하는 부분 (원인이 호출부에 있는 경우가 많아요)
  • "무엇을 기대했고, 실제로 무엇이 일어났는가" 한 줄

2) 단계별로 진행하기

맥락과 함께 한 번에 질문

아래 프롬프트의 대괄호만 내 상황으로 바꿔 그대로 붙여넣습니다.

아래 코드에서 이런 에러가 나. 원인을 찾아주고, 고친 코드와 '왜 그랬는지'를 설명해줘. 같은 실수를 막는 체크 포인트도 한 줄. 언어/환경: [예: Python 3.11] 기대한 동작: [무엇을 하려 했는지] 실제 동작: [무엇이 일어났는지] 에러 메시지(전문): [스택 트레이스 붙여넣기] 코드(문제 부분 + 호출부): [코드 붙여넣기]

원인 설명을 꼭 같이 받기

고친 코드만 받지 말고 "왜"를 이해해야 다음에 안 틀립니다. 설명이 빈약하면 이렇게 이어 묻습니다 → "방금 원인을 초보도 알게 한 단계씩 풀어줘."

직접 실행해 검증

받은 코드를 그대로 돌려봅니다. 또 막히면 새로 뜬 에러를 그대로 다시 붙여 ②번부터 반복합니다. 보통 2~3회면 잡힙니다.

재발 방지 한 줄로 마무리

"같은 실수를 막는 체크 포인트 한 줄과, 이 패턴이 코드에 또 있는지 알려줘." — 한 번의 디버깅을 코드 전체 점검으로 확장합니다.

6. 스프레드시트·Excel 심화

엑셀·구글시트는 "말로 설명하면 수식으로" 바꿔주는 AI와 궁합이 가장 좋은 도구입니다. 수식 작성·해석부터 데이터 정리, 피벗·대시보드, 매크로까지. 버전(엑셀/구글시트)과 실제 열 구조를 알려주는 것이 정확도의 핵심입니다.

"이런 계산을 하고 싶다"를 말로 설명하면 바로 쓸 수식으로 옮겨줍니다.

환경과 구조

엑셀인지 구글시트인지, 데이터가 어느 열(A·B…)에 있는지 알려줍니다.

원하는 계산 설명

붙여넣고 확인

받은 수식을 한 칸에 넣어 결과를 확인하고, 안 맞으면 예시를 주며 다시 요청합니다.

엑셀 수식을 만들어줘. - 환경: [예: Microsoft Excel 365 / 구글시트] - 데이터: [예: A열 날짜, B열 지역, C열 매출] - 하고 싶은 것: [예: '서울' 지역의 이번 달 매출 합계] - 빈 칸이나 오류가 있어도 깨지지 않게 처리해줘 - 수식과 함께, 각 부분이 무슨 역할인지 한 줄 설명도.
📝 입력
Excel 365 / A열 날짜, B열 지역, C열 매출 / '서울'의 이번 달 매출 합계
✨ 결과

수식
=SUMIFS(C:C, B:B, "서울", A:A, ">="&EOMONTH(TODAY(),-1)+1, A:A, "<="&EOMONTH(TODAY(),0))

설명 · SUMIFS로 '서울'이면서 날짜가 이번 달 1일~말일인 행의 매출만 더합니다. EOMONTH로 이번 달의 시작·끝 날짜를 자동 계산합니다.

수식 설명 · 오류 디버깅 ChatGPT 입문

남이 만든 복잡한 수식이 뭘 하는지, 왜 #REF!·#VALUE! 오류가 나는지 풀어줍니다.

아래 수식이 (1) 무슨 일을 하는지 쉽게 설명하고 (2) 지금 오류가 나는데 원인과 고친 수식을 알려줘. 환경: [엑셀/구글시트] 오류: [예: #REF! / #VALUE! / 결과가 0] 수식: [수식 붙여넣기] 원하는 결과: [무엇이 나와야 하는지]

날짜 통일·중복 제거·열 분리/병합 등 규칙을 적용해 정리합니다.

수식 vs 일괄

시트에 넣을 수식으로 받거나, 파일을 올려 정리된 결과 파일로 받습니다.

아래 표를 깨끗하게 정리해줘. - 날짜는 모두 YYYY-MM-DD로 통일 - 전화번호는 010-0000-0000 형식으로 - 이름 앞뒤 공백 제거, 중복 행 제거 - '이름' 열을 성/이름 두 열로 분리 정리 방법은 (A) 시트에 넣을 수식과 (B) 정리된 결과 표, 둘 다 보여줘. [지저분한 표 붙여넣기]

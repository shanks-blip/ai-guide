엑셀 수식·함수 도움 ChatGPT 입문

엑셀에서 이걸 하고 싶어. 수식을 알려주고, 각 부분이 무슨 뜻인지 한 줄씩 설명해줘. 가능하면 더 쉬운 대안 함수도 같이. 하고 싶은 것: [예: A열에 부서, B열에 금액이 있을 때 부서별 합계를 구하고 싶어] 내 엑셀 버전: [예: Microsoft 365 / 구버전]
📝 입력
A열 부서, B열 금액일 때 '영업팀' 합계를 구하고 싶어. Microsoft 365.
✨ 결과

수식 · =SUMIF(A:A, "영업팀", B:B)

설명 · A:A = 부서가 적힌 열에서 찾기 · "영업팀" = 찾을 조건 · B:B = 더할 금액 열.

대안 · 조건이 여러 개면 =SUMIFS(B:B, A:A, "영업팀", C:C, "2분기") 처럼 SUMIFS를 쓰세요.

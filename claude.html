나는 비개발자야. 매주 받는 영수증 사진 수십 장을 항목별로 묶어 엑셀로 정리하고 싶어. Claude 챗, Claude Cowork, Claude Code 중 무엇이 맞을지 이유와 함께 한 가지만 추천해줘.

기초 — Claude 챗 시작하기 입문

업데이트 · 2026-06-18

Anthropic이 개발한 LLM 기반 AI 어시스턴트입니다. 도움이 되고, 정직하며, 안전한 대화를 설계 원칙으로 삼습니다. 가입 후 바로 대화를 시작할 수 있습니다.

접속

웹 브라우저는 claude.ai ↗로 접속. 데스크톱·모바일 앱은 claude.com/download ↗에서 다운로드. macOS, Windows, iOS, Android 모두 지원합니다.

가입

이메일 또는 Google·Apple 계정으로 가입합니다. 만 18세 이상, 서비스 지원 국가에서 이용 가능. 무료 플랜은 하루 사용량 제한 내에서 핵심 기능을 사용할 수 있습니다.

첫 대화

채팅창에 자연스러운 한국어로 질문하면 됩니다. /를 입력하면 명령어 목록, + 버튼으로 파일 첨부가 가능합니다. 역할·말투·출력 형식도 요청하면 됩니다.

모델·설정 선택

입력창 근처 드롭다운에서 모델과 확장 사고(깊이 생각) 토글을 선택할 수 있습니다. 복잡한 문제는 확장 사고를 켜면 더 정교한 추론 과정을 보여줍니다. 한국어 답변은 "항상 한국어로 답해줘"라고 지정하면 됩니다.

입력 — 역할을 주고 구체적으로
너는 5년 경력 UX 디자이너야. 우리 팀이 만들고 있는 온보딩 화면의 사용자 흐름을 리뷰해줘. 주요 마찰 포인트와 개선 제안을 3가지씩 한국어로 알려줘.
결과

주요 마찰 포인트 3가지

  • 단계 수 과다 — 가입 단계가 7단계로 이탈률 상승 원인
  • 에러 메시지 불명확 — '잘못된 입력'만 표시되어 사용자가 무엇을 고쳐야 할지 모름
  • 진행률 미표시 — 현재 몇 단계 중 몇 단계인지 알 수 없어 불안감 유발

개선 제안 3가지

  • 소셜 로그인 우선 배치로 3단계 이내 완료 가능하도록 단순화
  • 인라인 실시간 유효성 검사 추가 및 구체적인 에러 문구 작성
  • 상단 스텝 인디케이터 + 완료 시 간단한 성취 애니메이션으로 동기 부여

꼭 쓰는 핵심 기능

업데이트 · 2026-06-18

Claude 챗을 제대로 활용하게 해주는 4가지 기능입니다. 각 기능에 실제 사용 예시를 함께 담았습니다.

Projects — 맥락이 이어지는 작업 공간

프로젝트 안에 파일·대화·사용자 지침을 함께 묶으면 Claude가 이전 대화를 기억하면서 작업을 이어갑니다. 반복되는 업무(주간 보고서, 제품 문서 작성, 팀 공유 작업)에 특히 유용합니다.

  • 회사 브랜드·말투 지침을 한 번만 등록하면 매번 설명 불필요
  • 관련 PDF·스프레드시트를 모아두면 참조 자동화
  • 팀원과 프로젝트 공유 가능(Team·Enterprise 플랜)

Artifacts — 살아있는 결과물

코드·HTML·React 컴포넌트·Markdown 문서·SVG 도표를 생성하면 오른쪽 패널에서 실시간 미리보기가 됩니다. 직접 편집하거나 링크로 바로 공유도 가능합니다.

가장 많이 쓰이는 활용: 할 일 웹앱 즉시 만들기

입력 — Artifacts 활용 (할 일 웹앱)
할 일 목록 웹앱을 단일 HTML 아티팩트로 만들어줘. 기능: 항목 추가·삭제·완료 체크, 우선순위(높음/보통/낮음) 선택, 다크모드 토글. 깔끔한 디자인으로.
결과

오른쪽 Artifacts 패널에 즉시 실행되는 완성된 할 일 앱이 생성됩니다. 추가/삭제/체크 모두 작동하며 다크모드 버튼도 달려 있습니다.

이후 "우선순위별로 정렬 추가해줘", "완료한 항목 숨기기 버튼 넣어줘" 등으로 반복 수정 가능합니다.

파일 업로드 · 웹 검색 · 커넥터(MCP)

입력창 + 버튼으로 PDF·이미지·Excel·CSV 등을 첨부해 분석·요약·변환할 수 있습니다. 웹 검색 토글을 켜면 최신 정보를 실시간으로 가져옵니다. 커넥터(MCP)를 연결하면 Google Drive·Slack 등 외부 도구에도 접근합니다.

  • PDF 보고서 → 핵심 요약·표 추출
  • 이미지 첨부 → 내용 분석·설명 생성
  • 웹 검색 → 오늘 환율·최신 뉴스 요약
  • Google Drive 커넥터 → 클라우드 파일 직접 참조
입력 — PDF 분석
[보고서.pdf 첨부] 이 PDF 보고서를 핵심 요점 5가지와 즉시 실행 가능한 액션 아이템 표로 정리해줘. 표는 '담당자 / 기한 / 우선순위' 열로 구성해줘.
결과

핵심 요점 5가지

  • Q2 매출 전분기 대비 18% 증가, 디지털 채널이 견인
  • 고객 재구매율 62%로 목표치 초과 달성
  • 물류 비용 증가가 수익률 하락 주요 원인
  • 신제품 라인 출시 계획 지연 — 공급망 이슈
  • 북미 시장 확장 기회 확인, 3분기 진입 검토 필요

액션 아이템 표 (Artifact에서 미리보기 가능)

담당자·기한·우선순위 열이 포함된 표가 Artifacts 패널에 생성됩니다.

Claude Cowork 시작하기 중급

업데이트 · 2026-07-07

Cowork는 Claude Code의 자율 작업 능력을 데스크톱 앱 안으로 가져온 에이전트 모드입니다. 일반 챗과 달리 Claude가 내가 허용한 폴더의 파일을 직접 읽고·수정·생성하며, 여러 단계 작업을 끝까지 완수합니다. 프롬프트를 잘 쓰는 것이 아니라 결과물을 직접 받는 방식입니다.

Cowork 준비물
① Claude 데스크톱 앱 (macOS · Windows · Linux 베타) — claude.com/download ↗
② 유료 구독 — Pro·Max·Team·Enterprise 플랜
③ 작업 중 앱이 열린 상태로, 컴퓨터가 켜져 있어야 합니다

설치부터 첫 Cowork 실행까지

데스크톱 앱 설치 또는 업데이트

claude.com/download ↗에서 macOS 또는 Windows용 앱을 받습니다. 이미 설치된 경우 최신 버전으로 업데이트하세요.

유료 구독 확인

Cowork는 Pro 이상 유료 플랜에서 사용 가능합니다. 앱 설정 → 구독에서 플랜을 확인하거나 업그레이드하세요.

Cowork 탭 선택

앱 상단 모드 선택기에서 Cowork 탭을 클릭합니다. Chat · Code · Cowork 세 탭이 나란히 있습니다.

작업 설명 + 폴더 접근 허용

원하는 작업을 입력창에 설명합니다. Claude가 필요한 폴더 접근을 요청하면 허용합니다. 파일 영구 삭제는 항상 별도 승인이 필요하니 안심하세요.

계획 검토 후 실행 방식 선택

Claude가 제시한 작업 계획을 검토하고 실행합니다. "단계마다 승인"(매 단계 확인 후 진행) 또는 "자동 실행"(끝까지 자동 완료)을 선택할 수 있습니다.

Cowork가 특별한 이유 — 핵심 능력

내 파일 직접 열고 저장업로드·다운로드 없이 로컬 파일을 읽고 수정·생성
여러 서브에이전트 병렬복잡한 일을 잘게 나눠 동시에 처리
완성형 산출물수식이 살아있는 엑셀·PPT·서식 문서로 바로 저장
예약 작업 자동 실행/schedule로 매일·매주 정해진 시간에 반복 수행

일하는 방식과 권한

작업을 맡기면 Cowork는 ① 요청을 분석해 계획을 세우고 ② 필요하면 여러 서브에이전트로 나눠 ③ 코드는 컴퓨터 안의 격리된 가상 환경(VM)에서 실행한 뒤 ④ 결과 파일을 폴더에 직접 저장합니다. 실행 방식은 두 가지 권한 모드로 고를 수 있습니다. 단계마다 승인은 매 동작 전에 멈춰 확인받는 안전한 방식이고, 자동 실행은 멈추지 않고 끝까지 진행하는 빠른 방식입니다. 어느 모드든 파일을 영구 삭제할 때는 항상 별도 승인을 받으니 안심하세요.

한 번 설정해 두면 매번 편해요 — 전역 지침 (설정 → Cowork)
역할·말투·산출물 형식을 미리 적어 두면 모든 작업에 자동으로 적용됩니다.
나는 마케팅팀 실무자야. 결과 문서는 항상 한국어 존댓말로 쓰고, 핵심 요약을 맨 앞에 3줄로 넣어줘. 표는 가능하면 .xlsx로 저장하고, 파일 이름은 YYYYMMDD_주제 형식으로 지어줘.
→ 결과
이후 모든 Cowork 작업이 이 규칙을 따릅니다. 매번 형식을 다시 설명할 필요가 없어 결과물이 한결 일관돼집니다.
자리를 비워도 돼요 — 모바일로 맡기기 (Pro·Max)

Cowork 실전 예시 중급

업데이트 · 2026-06-18

Cowork가 빛나는 순간은 반복적이고 여러 단계가 필요한 파일 작업입니다. 바로 붙여넣을 수 있는 예시 프롬프트를 모았습니다.

다운로드 폴더 자동 정리

수개월치 다운로드 폴더를 종류·날짜별로 자동 분류합니다. 계획 먼저 보여주기를 요청해 실행 전에 검토하세요.

내 다운로드 폴더를 스캔해서 파일을 종류(문서/이미지/영상/압축파일/기타)와 연도-월 기준으로 정리할 계획을 먼저 보여줘. 예상 폴더 구조와 이름 규칙, 중복 파일 처리 방법, 삭제가 필요할 파일 목록도 포함해줘. 내가 승인하면 그때 실행해줘.
결과 예시

Claude가 폴더를 스캔한 후 아래와 같이 계획을 제시합니다:

  • 2025-문서 — .pdf, .docx, .xlsx 파일 47개 이동 예정
  • 2026-이미지 — .png, .jpg, .gif 파일 123개 이동 예정
  • 중복 파일 — 12개 감지, 최신본 유지 후 나머지 삭제 대상(삭제는 별도 승인)

승인 후 실제 폴더 이동이 실행됩니다. 삭제는 추가 확인창이 뜹니다.

영수증 사진 → 경비 정산 엑셀

스마트폰으로 찍은 영수증 이미지들을 폴더에 모아두면, Cowork가 날짜·항목·금액을 추출해 합계 수식과 차트가 달린 엑셀 파일로 만들어줍니다.

이 폴더(receipts/)에 있는 영수증 사진 전부에서 날짜·가맹점명·항목·금액·부가세를 자동으로 추출해줘. 추출한 데이터를 열별로 정리하고, 월별 합계 행과 항목 분류별 파이 차트가 있는 경비 정산표(expense_2026.xlsx)로 저장해줘.
팁: 영수증이 흐릿하거나 글씨가 작은 경우 "인식이 불확실한 항목은 별도 시트에 표시해줘"라고 추가하면 나중에 검토하기 쉽습니다.

회의록 → 임원 보고 PPT

텍스트 회의록 파일만 있으면 슬라이드 구조를 Claude가 알아서 잡고 완성된 PowerPoint 파일을 만들어줍니다.

meeting_notes_0615.txt 파일을 읽어서 임원 보고용 PowerPoint를 만들어줘. 슬라이드 구성: 1장 표지(회의 제목·날짜·참석자), 2장 핵심 결론 요약, 3-6장 안건별 논의 내용·결정사항, 7장 액션 아이템 표(담당자·기한), 8장 다음 회의 일정. 전문적인 디자인으로 meeting_report.pptx 파일로 저장해줘.
결과 예시

Claude가 회의록을 분석한 후 meeting_report.pptx 파일을 지정 폴더에 생성합니다.

완료 후 "7장 액션 아이템 표의 담당자 열 폰트 크기를 키워줘"처럼 수정 요청도 바로 이어서 할 수 있습니다.

/schedule — 정기 자동 실행 예약

Cowork 입력창에 /schedule을 입력하면 작업을 정해진 시간에 자동 반복 실행하도록 예약할 수 있습니다.

/schedule 매주 월요일 오전 9시에 reports/ 폴더의 지난 주 데이터 파일들을 읽어서 주간 KPI 대시보드 보고서를 생성하고 weekly_report_YYYYMMDD.docx 형태로 저장해줘.
사용법: 입력창에 /schedule을 먼저 입력하면 예약 옵션 설정 화면이 나타납니다. 반복 주기(매일·매주·매월)와 실행 시간을 선택한 후 작업 내용을 입력합니다.
사용량 주의: Cowork는 일반 챗보다 사용량을 훨씬 빠르게 소모합니다. 간단한 질문이나 한 번의 생성은 일반 챗을, 여러 파일을 다루는 다단계 작업은 Cowork를 사용하는 것이 가장 효율적입니다. 사용량이 초과되면 다음 리셋 시까지 대기하거나 플랜 업그레이드를 검토하세요.

심화 — API · Claude Code · MCP 고급

업데이트 · 2026-06-18

Claude API 기초

Anthropic Console(platform.claude.com) ↗에서 API 키를 발급하고, Quickstart ↗ 문서를 따라 첫 메시지를 호출합니다.

핵심 엔드포인트는 Messages API로, 다중 턴 대화·시스템 프롬프트·도구 사용(tool use)·파일 입력을 지원합니다. 스트리밍, 확장 사고, 구조화 출력, 프롬프트 캐싱, 비전/PDF 처리도 사용할 수 있습니다.

시작 방법: Console → API Keys → 새 키 생성 → Quickstart 문서 참고 → 첫 curl 또는 Python 호출.

Claude Code — 에이전트 코딩 도구

코드베이스 전체를 이해해 파일 편집·명령 실행·Git 커밋·테스트까지 수행하는 개발자용 CLI 도구입니다. 터미널·VS Code·JetBrains 통합·웹(claude.ai/code ↗)에서 사용 가능합니다.

설치

macOS / Linux:

curl -fsSL https://claude.ai/install.sh | bash

Windows (PowerShell):

irm https://claude.ai/install.ps1 | iex

설치 후 원하는 프로젝트 폴더로 이동해 claude를 실행합니다. 프로젝트 루트에 CLAUDE.md 파일을 두면 프로젝트 지침이 자동 적용됩니다.

팁: CLAUDE.md에 코딩 컨벤션, 금지 명령, 빌드 방법 등을 적어두면 Claude Code가 프로젝트를 더 잘 이해합니다.

자세한 내용은 심화 활용 페이지에서 다룹니다.

MCP (Model Context Protocol)

Anthropic이 발표한 오픈 표준으로, AI 모델과 외부 도구·데이터 소스를 안전하게 연결하는 프로토콜입니다. USB-C처럼 어떤 도구든 표준 방식으로 Claude에 연결할 수 있습니다.

Google Drive 커넥터클라우드 파일 직접 읽기·쓰기
Slack 커넥터채널 메시지 읽기·전송
GitHub 서버PR·이슈·코드 연동
웹 검색·스크래핑실시간 정보 가져오기

MCP 서버 목록과 사양은 modelcontextprotocol.io ↗와 공식 서버 모음 ↗에서 확인하세요. 자세한 연결 방법은 심화 활용 페이지를 참고하세요.

자주 묻는 질문

업데이트 · 2026-06-21
무료로도 Claude를 쓸 수 있나요?

네, 가능합니다. Claude 챗은 무료 플랜이 있어 일정 사용량까지 대화·분석·아티팩트 생성 등 핵심 기능을 사용할 수 있습니다. 다만 하루 사용량에 제한이 있으며, 더 많이 사용하거나 Cowork·고급 기능이 필요하면 유료 플랜으로 업그레이드해야 합니다.

Cowork는 꼭 유료 구독이 필요한가요?

네, Cowork는 Pro·Max·Team·Enterprise 유료 플랜에서만 사용할 수 있습니다. 또한 데스크톱 앱(macOS 또는 Windows) 설치가 필수입니다. 웹 브라우저나 모바일 앱에서는 Cowork 모드를 이용할 수 없습니다.

한국어 대화가 잘 되나요?

매우 잘 됩니다. 한국어로 질문하면 자연스러운 한국어로 답변합니다. 항상 한국어로 받고 싶다면 처음에 "앞으로 모든 답변은 한국어로 해줘"라고 지시하거나, Projects 사용자 지침에 한 번만 등록해두면 됩니다. 다만 아주 전문적인 최신 한국어 자료는 영어 자료보다 학습량이 적을 수 있으니, 중요한 사실은 꼭 확인하세요.

내 파일·데이터가 Claude에 학습되나요? 안전한가요?

Anthropic의 공식 정책에 따르면 API 사용자와 Pro 이상 유료 플랜의 데이터는 모델 학습에 기본적으로 사용되지 않습니다. Cowork는 내가 허용한 폴더에만 접근하며, 허용하지 않은 파일에는 접근할 수 없습니다. 파일 영구 삭제는 항상 별도 승인이 필요합니다. 민감한 업무 파일의 경우 회사 정책과 Anthropic 이용약관·개인정보 처리방침을 반드시 확인하세요.

일반 챗과 Cowork의 가장 큰 차이가 뭔가요?

일반 챗은 Claude가 텍스트로 답변을 줍니다. 파일을 만들거나 수정하려면 내가 직접 복사해서 작업해야 합니다. 반면 Cowork는 Claude가 내 컴퓨터 파일에 직접 접근해 실제로 파일을 읽고·수정하고·생성합니다. 예를 들어 "회의록을 PPT로 만들어줘"라고 하면, 챗은 PPT 내용을 텍스트로 제안하고 직접 만들어야 하지만, Cowork는 실제 .pptx 파일을 완성해 폴더에 저장해줍니다.

Artifacts는 어떻게 공유하나요?

Artifacts 패널 상단에 공유 버튼(링크 아이콘)이 있습니다. 클릭하면 해당 Artifact를 볼 수 있는 링크가 생성됩니다. 이 링크를 팀원이나 고객에게 공유하면 별도 가입 없이 바로 볼 수 있습니다. HTML 앱이나 대화형 차트도 링크 하나로 공유가 됩니다.

Cowork 중 앱을 끄면 어떻게 되나요?

작업 중에는 데스크톱 앱이 열려 있고 컴퓨터가 켜져 있어야 합니다. 앱을 닫거나 컴퓨터가 잠금·종료되면 진행 중인 작업이 중단될 수 있습니다. 긴 배치 작업이나 예약 작업(/schedule)은 완료될 때까지 앱을 켜두는 것을 권장합니다.

예: 요금이 헷갈릴 때

“무료로 어디까지 되고, 언제 유료가 필요한지 표로 알려줘.”

무료=일상 대화·분석·아티팩트, 유료=Cowork·대용량·높은 사용량 식으로 정리해 줍니다.
